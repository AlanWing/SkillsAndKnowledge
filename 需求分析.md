# 登录
请求URL:  management/instructor/login   
参数:  username,password   
请求方式 : POST  
```json   
    成功
    {
        "code":0,
        "data":"登录成功"
    }
    
    失败 
    {
        "code":1,
        "data":"用户认证失败"
    }
```
